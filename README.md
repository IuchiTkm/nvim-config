## キーマップ一覧

### ファイルツリー
| キー | 動作 |
|------|------|
| `Ctrl+b` | ファイルツリーの開閉 |

### タブ（バッファ）
| キー | 動作 |
|------|------|
| `Shift+l` | 右のタブへ |
| `Shift+h` | 左のタブへ |
| `\w` | 現在のタブを閉じる |

### ウィンドウ分割
| キー | 動作 |
|------|------|
| `\d` | 上下分割 |
| `\r` | 左右分割 |
| `\x` | ウィンドウを閉じる |
| `\h` / `\j` / `\k` / `\l` | ウィンドウ間の移動 |
| `\z` | ズーム切り替え |

### Copilot
| キー | 動作 |
|------|------|
| `\ccq` | バッファ内容を使ってCopilotにチャット |

### Markdown
| キー | 動作 |
|------|------|
| `\mp` | Markdownプレビューのオン/オフ |

## プラグイン詳細

### nvim-tree

ファイルツリーを右側に表示するプラグイン。

**ツリー内の操作**

| キー | 動作 |
|------|------|
| `Enter` / `o` | ファイルを現在のウィンドウで開く |
| `t` | 新しいタブで開く |
| `T` | 新しいタブで開く（フォーカスはツリーのまま） |
| `a` | 新規ファイル/ディレクトリ作成（末尾に `/` でディレクトリ） |
| `r` | リネーム / 別フォルダへ移動（パスごと書き換え） |
| `d` | 削除 |
| `x` | カット |
| `c` | コピー |
| `p` | ペースト |
| `R` | ツリーを更新 |
| `H` | 隠しファイルの表示切り替え |
| `q` | ツリーを閉じる |

---

### bufferline

開いているバッファをVSCodeのようにタブ表示するプラグイン。

| キー | 動作 |
|------|------|
| `Shift+l` | 右のタブへ |
| `Shift+h` | 左のタブへ |
| `\w` | 現在のタブを閉じる |

---

### lualine

ステータスラインを見やすく表示するプラグイン。設定不要で自動的に有効になる。

---

### coc.nvim

LSP（Language Server Protocol）による補完・定義ジャンプなどを提供するプラグイン。

**主なコマンド**

| コマンド | 動作 |
|----------|------|
| `:CocInstall coc-<name>` | 言語サーバーのインストール |
| `:CocList extensions` | インストール済み拡張の一覧 |
| `:CocUpdate` | 拡張の更新 |

**主なキー（デフォルト）**

| キー | 動作 |
|------|------|
| `gd` | 定義へジャンプ |
| `gr` | 参照一覧 |
| `K` | ホバードキュメント表示 |
| `Tab` | 補完候補を選択 |

---

### CopilotChat

GitHub Copilot とチャット形式で対話できるプラグイン。

| キー | 動作 |
|------|------|
| `\ccq` | バッファ内容を添付してチャット入力 |

**主なコマンド**

| コマンド | 動作 |
|----------|------|
| `:CopilotChat` | チャットウィンドウを開く |
| `:CopilotChatClose` | チャットウィンドウを閉じる |
| `:CopilotChatReset` | チャット履歴をリセット |

---

### render-markdown

Markdownファイルをバッファ内でリッチ表示するプラグイン。`.md` ファイルを開いたときのみ有効。

| キー | 動作 |
|------|------|
| `\mp` | プレビューのオン/オフ切り替え |

---

### winresizer

`Ctrl+e` でウィンドウリサイズモードに入るプラグイン。

| キー | 動作 |
|------|------|
| `Ctrl+e` | リサイズモード開始 |
| `h` / `j` / `k` / `l` | ウィンドウ幅・高さを変更 |
| `Enter` | リサイズモード終了 |
| `q` | キャンセル |
